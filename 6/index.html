C++講習会2022

繰り返し

目標

同じ処理を決めた回数繰り返すことができるようになる。

語句説明

カウンター

任意の回数繰り返すとき、何回繰り返したか保存しておくため使われる。 慣習的に、カウンターの変数名はi以降の1文字の小文字アルファベットになることが多い。

サンプル１

解説

for(;;)

for文は、&lbrace;&rbrace;の中にある処理を繰り返す文です。

for(変数の宣言初期化; 条件式; 変数の変化) &lbrace; 処理 &rbrace;

変数の宣言では、繰り返すときのカウンターにする変数を宣言します。

for文は、指定した条件式がtrueの間繰り返されます。

変数の変化は主にカウンターを増加、減少させるときに使います。

for(int i = 0; i < 5; i++) &lbrace; // iが1ずつ増加していく。 // このなかで繰り返したい処理を行う。 &rbrace;
中身
std::cout << "Hello_World!!  " << 1 << std::endl;
から
std::cout << "Hello_World!!  " << 5 << std::endl;
までと、
for(int i = 1; i < 6; i++) &lbrace; std::cout << "Hello_World!!  " << i << std::endl; &rbrace;
の意味は全く変わらない。

サンプル２

解説

while()

while文は、&lbrace;&rbrace;の中にある処理を繰り返す文です。

while(条件式) &lbrace; 処理 &rbrace;

whileは、指定した条件式がtrueの間繰り返されます。 主に何回繰り返すか分からない処理に使われます。

中身
while(input != 0) &lbrace; // inputが0以外のときtrue。 // このなかで繰り返したい処理を行う。 &rbrace;

入力が0でない間、入力された数字を加算することを繰り返します。

課題

  • 1から50000までの和を出力する。
  • 要提出 while文を使い数字当てゲームを作る。数字を当てるまで繰り返す。 入力された数ではなかった場合その数より上か下か教える。
  • 要提出 九九列挙(クリックで説明を表示)
